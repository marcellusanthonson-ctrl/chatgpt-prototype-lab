# PAT-MA-001 — Codex Is Executor Only

Status: `APPROVED`
Date: `2026-06-28`

## Rule

Codex no puede iniciar, decidir, ampliar alcance ni modificar archivos, estado, stack, Git o repositorios sin una instrucción directa, explícita y acotada emitida por ChatGPT bajo autorización del usuario.

## Clarification

Un brief aprobado, fingerprint válido, README, `AGENTS.md`, `status.json` o preflight PASS son precondiciones, no autorización autónoma.
